# Motion-Software
Software and manuals for Leadshine, JMC, Stepperonline, RIntelligent.

### This is not being updated 
Please see the manufacturers site for new software. These files were downloaded from manufacturer sites, but I cannot verify integrity. Make sure to virus scan everything.



## Brands 
When using RS232 / RS485 for communication to a servo / stepper, it is possible to use software from a different manufacturer than your hardware. Especially between LeadShine and StepperOnline products (they seem to resell some identical products)  RIntelligent's software works quite well across different motors.
